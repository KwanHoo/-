## 🌱 풀이 과정 습관화 
  1. 문제를 읽고 요구사항을 이해한다.
  2. 문제를 내가 이해하기 쉽게 요약 및 정리한다.
  3. 문제를 어떻게 해결할지 계획한다. (해결 방법, 알고리즘과 자료구조 선택, 언어선택)
  4. 계획을 검증 (시간복잡도 고려, 효율성 분석)
  5. 프로그램으로 작성
  6. 회고 💬
     (내 풀이방법을 돌아보고 기록하며, 한번에 풀지 못한 문제는 오답의 원인을 분석하고 기록하자)
